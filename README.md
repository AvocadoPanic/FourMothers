# Four Mothers, Four Fathers

A field-guide instrument adapting Christine Ann Lawson's typology of borderline mothers and the men who marry them.

**Live:** https://avocadopanic.github.io/FourMothers/

## About

An interactive self-assessment built around the four maternal patterns Lawson identifies — the **Waif**, the **Hermit**, the **Queen**, and the **Witch** — and the four corresponding fathers — the **Frog-Prince**, the **Huntsman**, the **King**, and the **Fisherman**.

Each side presents 36 forced-choice items with:

- a confidence weighting per answer (*Loosely / Mostly / Exactly*, contributing 1× / 2× / 3× to the chosen type)
- an explicit *None of these describes me* option per question, tallied separately so the result can be **no strong match** rather than a forced winner
- a radar profile with public-domain fairy-tale portraits at each vertex

This is an instrument for **naming a pattern**, in oneself or in someone close. It is not a diagnostic tool.

## Not a diagnostic instrument

This page is informational. It does not diagnose Borderline Personality Disorder or any other condition. If you are concerned about your own mental health or someone else's, talk to a licensed clinician.

## Source material

The typology is adapted from:

> Christine Ann Lawson, Ph.D. *Understanding the Borderline Mother: Helping Her Children Transcend the Intense, Unpredictable, and Volatile Relationship.* Jason Aronson, 2000.

Lawson's book is not distributed with this repository.

## Running it

Nothing to build. `index.html` is self-contained and includes all CSS and JavaScript inline. The eight images in `img/` are referenced by relative path. Open `index.html` directly in a browser, or serve the repository root over any static HTTP server. GitHub Pages serves it at the URL above.
